# EB Monorepo with Turborepo

## Prerequisites

Node.js v22+

## Usage

Install dependencies:

```bash
npm install
```

Run app:

```bash
npm run dev
```

front-end will by default run on [http://localhost:3000](http://localhost:3000).

back-end will by default run on [http://localhost:4000](http://localhost:4000).

more information of each app is available in the app readme.
